# 🚀 Network Port Scanner GUI

A lightweight TCP port scanner with a graphical user interface built using Python and Tkinter.

---

## ✨ Features

- Simple 3-field interface (Host, Start Port, End Port)
- Multi-threaded scanning (up to 500 threads)
- Service identification (HTTP, HTTPS, SSH, etc.)
- Real-time progress bar
- Stop scan anytime
- Save results to `.txt`
- Cross-platform (Windows, macOS, Linux)

---

## 📌 Requirements

- Python 3.7+
- Tkinter (comes pre-installed)
